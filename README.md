# Duo Conceito | Móveis Planejados Premium

![Duo Conceito](assets/logo.png)

> Transforme sua casa no lar dos seus sonhos com móveis planejados de design exclusivo, qualidade premium e condições especiais.

Este é o repositório oficial da **Duo Conceito**, uma plataforma web moderna projetada para apresentar ambientes planejados de alto padrão e facilitar o contato com clientes através de um sistema de briefing inteligente.

---

## 🚀 Funcionalidades

- **Design Premium & Responsivo:** Interface elegante, inspirada em tendências modernas de arquitetura e design de interiores.
- **Galeria de Ambientes Categorizada:** Filtros dinâmicos para explorar Cozinhas, Salas, Dormitórios, Banheiros e Lavanderias.
- **Briefing Inteligente (Multi-step Form):** Formulário em etapas para capturar necessidades específicas do cliente (tipo de imóvel, ambientes, investimento).
- **Integração Direta com WhatsApp:** Envio automático dos dados do briefing diretamente para a equipe comercial via WhatsApp API.
- **Animações Fluidas:** Experiência de usuário aprimorada com animações de revelação ao scroll (Intersection Observer).
- **FAQ Interativo:** Seção de dúvidas frequentes com sistema de acordeão.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido focando em performance, SEO e código limpo:

- **HTML5:** Estrutura semântica para melhor acessibilidade e indexação.
- **CSS3 Moderno:** 
  - Variáveis CSS (Custom Properties) para consistência visual.
  - CSS Grid e Flexbox para layouts complexos e responsivos.
  - Design focado em conversão e estética "Premium".
- **JavaScript (Vanilla):**
  - Lógica de formulário multi-etapas.
  - Sistema de filtros da galeria sem bibliotecas externas.
  - Intersection Observer API para otimização de performance em animações.
- **Phosphor Icons:** Biblioteca de ícones moderna e leve.
